Lucene is an open source toolkit of full-text search engine, that it is not a full text search engine, but a full text search engine architecture provides a complete query engine and index engine, part of the text analysis engine (in English and German two west languages). Lucene is for software developers to provide a simple and easy to use tool kit, in order to facilitate the realization of the target system in full-text retrieval function, or is this as the foundation established a complete full-text search engine.
